PotgCraft 服务器介绍 ​

PotgCraft 是一个长期运行的公益 Minecraft 服务器，以 生存 · 建筑 · 生电 · 原版技术探索 · 社交为主

INFO

在这里，世界不会轻易清档，你的努力与创造将被长期保留。

核心目标 ​

  • 建立一个长期存在的世界
  • 鼓励玩家探索原版机制与技术玩法
  • 保留玩家建筑和大型工程
  • 提供合理增强的玩法，而不破坏平衡

TIP

与快餐服不同，我们不追求短期人气，而是注重长期生态沉淀。

特色玩法 ​

原版生存 ​

  • 保持原版生存循环：采矿、建造、探险、社交
  • 主世界内不使用破坏平衡的插件
  • 支持红石与生电实验

技术与工程 ​

  • 玩家可以搭建红石装置、生电工程
  • 提供轻量数据包增强，辅助建筑与技术玩法
  • 鼓励玩家探索复杂机制，但对极端卡服机器会有限制

WARNING

破坏服务器稳定性或性能的行为可能会被限制。

内置自定义合成配方 ​

服务器内通过数据包提供了若干自定义合成，主要用于 娱乐性、建筑辅助或技术实验，

查看自定义合成

请参考 自定义合成表 页面获取完整信息。

加入服务器 ​

IP 与版本 ​

  • 服务器地址：minecraft.Potg.org
  • 版本要求：Minecraft Java 版 1.20+ 推荐1.21.11
  • 客户端要求：推荐下载服务器的专属客户端然后自行调整，支持 JEI 查看自定义合成
  • 正版验证：已开启(非正版用户加群添加白名单)

新人提示 ​

TIP

如果你希望你的单机建筑或作品迁移到服务器，可联系管理员申请搬迁。

WARNING

请遵守规则，尊重世界和其他玩家的创造。违规行为可能会受到限制。
